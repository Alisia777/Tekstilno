# Apps Script для записи отчётов в Google Sheet

## Самый быстрый сценарий

1. Откройте Google Sheet, куда должны падать отчёты.
2. Нажмите **Extensions → Apps Script**.
3. Удалите примерный код и вставьте содержимое `Code.gs`.
4. Сохраните проект.
5. Нажмите **Deploy → New deployment**.
6. Выберите тип **Web app**.
7. Execute as: **Me**.
8. Who has access: **Anyone** — это самый простой вариант для запуска без логина.
9. Нажмите **Deploy** и скопируйте URL, который заканчивается на `/exec`.
10. Вставьте этот URL в `config.js` в поле `appsScriptUrl`.

## Какую таблицу использует код

По умолчанию код пишет в лист `DailyReports`.

- Если скрипт привязан к Google Sheet, оставьте `SPREADSHEET_ID` пустым.
- Если скрипт standalone, вставьте ID таблицы в `SPREADSHEET_ID`.

## Что попадёт в таблицу

- дата отчёта;
- имя менеджера;
- отдел;
- статус дня;
- KPI;
- текстовые блоки отчёта;
- локальный timestamp отправки;
- URL страницы;
- timezone браузера.

## Важно

Если захотите встроить таблицу на сайт через `googleSheetEmbedUrl`, таблицу нужно будет
**опубликовать в веб**, а это делает read-only просмотр публичным по ссылке.
Если нужна приватность — не встраивайте, а просто открывайте обычную Google Sheet по кнопке.
