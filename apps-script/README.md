# Apps Script для сайта «Тетрадь менеджера»

## Что делает скрипт

Сайт на GitHub Pages отправляет один большой `payloadJson` в Apps Script.  
Скрипт раскладывает его по трём листам:

- `DailyReports` — одна строка на один отчёт;
- `TaskStatusLog` — статусы задач из этого отчёта;
- `ManagerActionsLog` — записи из тетради действий.

## Как развернуть

1. Создайте Google Sheet.
2. Откройте **Extensions → Apps Script**.
3. Вставьте код из `Code.gs`.
4. Если скрипт не привязан к таблице, впишите `SPREADSHEET_ID`.
5. Нажмите **Deploy → New deployment → Web app**.
6. Выберите:
   - **Execute as:** `Me`
   - **Who has access:** `Anyone`
7. Скопируйте URL вида `/exec`.
8. Вставьте его в `config.js` в поле `appsScriptUrl`.

## Что важно

В этой версии фронт отправляет данные через скрытый iframe.  
Это сделано специально, чтобы GitHub Pages мог спокойно отправлять данные в Apps Script без попытки читать кросс-доменный JSON-ответ.

## Проверка

После деплоя откройте URL `/exec` в браузере.  
Если всё нормально, увидите простой экран со статусом `OK`.

После первой отправки отчёта в таблице автоматически появятся листы с заголовками.
