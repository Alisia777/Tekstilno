Как считается пакет

  1. Для WB и Ozon берётся общий список артикулов по площадке.
  2. Артикулы сортируются сначала по срочности комментария, потом по плановому вкладу.
  3. Список режется на пакеты по 20 артикулов на день.
  4. Внутри пакета менеджер видит «что сделать сегодня» и комментарий из исходного файла.
  5. План на день = план месяца / число рабочих дней месяца.
