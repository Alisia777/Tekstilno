Как считается пакет

  1. Берётся список артикулов менеджера.
  2. Для выбранного месяца они сортируются по маржинальному вкладу.
  3. Список режется на пакеты по 20 артикулов.
  4. Пакет меняется по рабочим дням.
  5. План на день = план месяца / число рабочих дней месяца.
