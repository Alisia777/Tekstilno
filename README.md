# manager-journal-site-v15

Обновление модуля Нины под кластерную матрицу WB / Ozon.

## Что изменено

- упрощена навигация: `Матрица / Заявки / Как считаем`;
- добавлен переключатель расчета оборачиваемости: `7 / 14 / 28 дней`;
- в матрице слева оставлены все артикула, сверху — кластеры и склады;
- показан `План месяца, шт` по артикулу и по каждому кластеру;
- внесение данных перенесено прямо в матрицу:
  - `В пути`
  - `Производство`
  - `Закупка`
  - `Дата прихода`
  - `Комментарий`
- кнопка `В заявку` открывает форму пополнения по нужному артикулу и кластеру;
- добавлен экспорт в Excel (`.xls`, открывается в Excel) по:
  - WB матрице
  - Ozon матрице
  - внесенным данным
  - журналу заявок

## Основные файлы

- `index.html` — основной вход в сайт
- `nina.html` — матрица Нины
- `nina.js` — логика матрицы, inline-ввод, экспорт Excel
- `nina.css` — верстка модуля Нины
- `data/nina-cluster-dashboard.json` — исходные данные кластеров

## Как обновлять

1. Заменить файлы в GitHub-репозитории содержимым этого архива.
2. Дождаться обновления Pages.
3. Сделать `Ctrl+F5`.
4. Открыть `nina.html`.

## Что хранится локально

Пока без Google backend данные сохраняются локально в браузере:
- внесенные данные по кластерам
- журнал заявок
- настройки горизонта оборачиваемости и целевых дней

Для переноса между устройствами:
- `Резерв JSON`
- `Импорт JSON`
- `Экспорт Excel`
