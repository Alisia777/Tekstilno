# Manager Journal Site v5

Обновлённая версия сайта под реальный процесс команды:

- Анастасия — WB
- Ирина Паламарук — Ozon
- Координатор поставок — отдельная роль без пакета артикулов

## Что поменялось

- Для пакета артикулов добавлен отдельный блок **«Что сделать сегодня»**
- В этот блок подтягивается **комментарий из исходных файлов**
- Приоритет теперь считается **по срочности комментария и статусу артикула**, а не только по цифрам
- Кнопка Google Sheet скрыта, пока вы не заполните ссылку в `config.js`
- Добавлен `config.js`, чтобы GitHub Pages не ругался на отсутствующий файл

## Источники комментариев

- WB: `Оцифровка в руб  План продаж WB 2026.xlsx`, лист `Александра`
- Ozon: `План продаж OZON 2026.xlsx`, листы `Даниил` и `Мир`

## Быстрый запуск

1. Заменить файлы в GitHub-репозитории содержимым этой папки
2. Подождать обновления GitHub Pages
3. Сделать жёсткое обновление страницы в браузере (`Ctrl+F5`)
4. При желании позже заполнить `appsScriptUrl` и `googleSheetUrl` в `config.js`

## Что редактировать вручную

- `data/daily-tasks.json` — ежедневные задачи по ролям
- `config.js` — подключение общей таблицы и Apps Script
