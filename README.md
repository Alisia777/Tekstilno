# Тетрадь менеджера по артикулам — GitHub Pages

Эта версия уже собрана под ваш реальный процесс:

- менеджер видит **свои 20 артикулов на день**;
- пакет крутится **по рабочим дням**;
- внутри есть **задачи на день**;
- есть **тетрадь действий**;
- есть **план / факт / отклонение** по заказам, марже и выручке;
- всё работает **локально в браузере**, даже пока Google Apps Script не развернут.

## Какие файлы важны

- `index.html` — страница сайта
- `styles.css` — стили
- `app.js` — логика сайта
- `config.js` — настройки
- `data/article-plan.json` — уже собранный план по артикулам
- `data/daily-tasks.json` — ежедневные задачи
- `data/manager-pack-2026-04-02.csv` — готовый CSV со стартовым пакетом на 02.04.2026
- `apps-script/Code.gs` — backend для Google Sheet

## Что менять руками каждый день

### 1. Если нужно обновить задачи на день
Откройте файл:

`data/daily-tasks.json`

Там можно менять:

- общие задачи для всех;
- задачи для конкретного менеджера;
- задачи на конкретную дату.

### 2. Если хотите изменить дату старта ротации
Откройте:

`config.js`

И поменяйте:

`cycleAnchorDate: "2026-04-02"`

Если хотите, чтобы первый пакет начинался с другой даты — просто замените её.

### 3. Если хотите подключить общую таблицу
В `config.js` заполните:

- `appsScriptUrl`
- `googleSheetUrl`

## Как залить в GitHub

1. Замените файлы в текущем репозитории этой новой версией.
2. Убедитесь, что GitHub Pages публикует сайт из корня ветки.
3. Откройте сайт по ссылке Pages.

## Как подключить Google Sheet позже

1. Создайте Google Sheet.
2. Откройте **Расширения → Apps Script**.
3. Вставьте код из `apps-script/Code.gs`.
4. Разверните как Web App.
5. Вставьте URL web app в `config.js` в поле `appsScriptUrl`.
6. В `googleSheetUrl` вставьте ссылку на рабочую таблицу.

Скрипт пишет в четыре листа:

- `ManagerReports`
- `TaskStatuses`
- `ArticleFacts`
- `ActionJournal`

## Что менеджер делает в этой версии

1. Выбирает себя и дату.
2. Видит свои 20 артикулов на день.
3. Проставляет факт по артикулу, статус и комментарий.
4. Отмечает ежедневные задачи.
5. Ведёт тетрадь действий.
6. Сохраняет отчёт.
