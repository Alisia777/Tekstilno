# Текстильно — сайт v11

Что нового:
- график **план / факт по дням** на вкладках WB и Ozon;
- переключатель **Заказы / Выручка**;
- переключатель периода **7 / 14 / 30 дней**;
- топ-5 отклонений теперь считает разрыв **за выбранный период**, а не только по одной точке.

Источники графика:
- **Ozon** — фактические дневные данные из `orders (44).csv` по дате `Принят в обработку`;
- **WB** — дневная линия пока построена по текущему **7-дневному среднему**, потому что в доступных файлах нет отдельной WB-выгрузки по дням.

Файлы данных:
- `data/article-plan.json` — план и текущие задачи по артикулам;
- `data/cluster-reference.json` — кластера и остатки для блока поставок;
- `data/history-data.json` — дневная история для графиков.

Что обновлять дальше:
- если нужен новый график Ozon — обновляется `data/history-data.json` на основе свежего `orders.csv`;
- если нужен настоящий дневной график WB — нужна отдельная выгрузка WB **по дням**.

Публикация:
1. Заменить содержимое репозитория файлами из этой папки.
2. Дождаться обновления сайта.
3. Сделать `Ctrl+F5`.
