# Текстильно · сайт v10

Что изменено в этой версии:

- нормальная верстка: меньше текста, компактнее карточки, чище таблицы;
- вкладки по ролям: `WB`, `Ozon`, `Поставки`, `Руководитель`;
- менеджеры заполняют только `статус` и `комментарий` по артикулам;
- блок `Темп к плану` добавлен отдельно по каждому менеджеру;
- блок `Топ-5 отклонений` вынесен отдельно от дневного пакета;
- по клику на артикул из `Топ-5 отклонений` сайт сразу открывает нужный пакет и подставляет поиск;
- план/факт по артикулам по-прежнему подтягивается автоматически из `data/article-plan.json`.

## Важно про график

График в этой версии показывает **траекторию пакета по текущему темпу**:
- синяя линия — план;
- красная линия — факт по текущему темпу.

Это сделано специально, чтобы быстро видеть отставание или опережение без ручного просмотра всех строк.

## Что править ежедневно

Обычно ежедневно править не нужно сам сайт.
Если меняется список артикулов, задачи или метрики, обновляется файл:

- `data/article-plan.json`

Если подключён Apps Script, то сданные пакеты и формы поставок будут храниться уже в общей таблице, а не только локально в браузере.

## Для GitHub Pages

Заливать в репозиторий нужно **всю папку целиком**:

- `index.html`
- `styles.css`
- `config.js`
- `app.js`
- `data/`
- `apps-script/` при необходимости

После обновления на GitHub Pages лучше сделать `Ctrl+F5`, чтобы не тянулся старый кэш.
