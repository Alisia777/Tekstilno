# Текстильно — сайт v12

В этой версии добавлена отдельная страница для Нины:

- `nina.html` — **Нина · WB / Ozon кластеры, заказ товара, внесение данных**

## Что нового

### 1) Отдельный модуль для Нины
- переключалка **WB / Ozon**;
- вкладки:
  - **Матрица**
  - **Запасы и оборачиваемость**
  - **Заказ товара**
  - **Внесение данных**
  - **Как считаем**

### 2) Матрица по кластерам
Слева:
- артикул / размер
- товар
- приоритет
- основной склад

Сразу после фиксированных колонок идет блок **ИТОГО**.

По каждому кластеру:
- заказы 7д
- план/день
- запас
- в пути
- производство
- закупка
- оборачиваемость
- рекомендуемая отгрузка

### 3) Формула расчета
1. Доля кластера = заказы 7д кластера / заказы 7д артикула  
2. Сезонный план / день = план дня текущего месяца × доля кластера  
3. Спрос / день для расчета = `max(факт по кластеру, сезонный план)`  
4. Доступно = запас + в пути + производство + закупка  
5. Реком. отгрузка = `max(0, спрос/день × целевые дни − доступно)`

### 4) Сезонность
Коэффициент сезона берется из месячного плана по артикулам:

`коэф. сезона = план/день текущего месяца ÷ средний план/день по доступным месяцам`

Есть ручной override в форме.

### 5) Формы
- **Быстрое внесение данных** — в пути / производство / закупка / target days / seasonality override
- **Заказ товара** — заявка на отгрузку / производство / закупку
- **Экспорт / импорт JSON**
- экспорт CSV по:
  - запасам
  - ручным данным
  - заявкам

## Публикация
1. Заменить содержимое репозитория файлами из этой папки.
2. Дождаться обновления сайта.
3. Открыть:
   - основной сайт: `index.html`
   - модуль Нины: `nina.html`
4. Сделать `Ctrl+F5`.

## Важно
Ручные данные на `nina.html` пока сохраняются **локально в браузере**.  
Чтобы переносить их между устройствами — используйте экспорт / импорт JSON.
