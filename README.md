# Текстильно — сайт v13

В этой версии усилен блок **Поставки** в основном сайте и сохранена отдельная страница для Нины:

- `index.html` — обновлённый таб **Поставки**: сводка по площадке, топ дефицитов, зона внимания, итоговая строка по артикулу
- `nina.html` — **Нина · WB / Ozon кластеры, заказ товара, внесение данных**

## Что нового

### 1) Поставки стали прозрачнее в основном сайте
- исправлен JS-блок, из-за которого карточки по выбранному артикулу могли оставаться пустыми;
- добавлена сводка по площадке: артикулы, кластерные строки, спрос/день, запас на МП, осн. склад, нужно к отгрузке;
- добавлен список **Топ дефицитов** с быстрым переходом к артикулу;
- добавлен список **Требует решения сегодня**;
- в таблице по артикулу появилась строка **ИТОГО**;
- выведены блоки **Почему артикул в фокусе** и **Что делаем по артикулу**.

### 2) Отдельный модуль для Нины
- переключалка **WB / Ozon**;
- вкладки:
  - **Матрица**
  - **Запасы и оборачиваемость**
  - **Заказ товара**
  - **Внесение данных**
  - **Как считаем**

### 3) Матрица по кластерам
Слева:
- артикул / размер
- товар
- приоритет
- основной склад

Сразу после фиксированных колонок идет блок **ИТОГО**.

По каждому кластеру:
- заказы 7д
- план/день
- запас
- в пути
- производство
- закупка
- оборачиваемость
- рекомендуемая отгрузка

### 4) Формула расчета
1. Доля кластера = заказы 7д кластера / заказы 7д артикула  
2. Сезонный план / день = план дня текущего месяца × доля кластера  
3. Спрос / день для расчета = `max(факт по кластеру, сезонный план)`  
4. Доступно = запас + в пути + производство + закупка  
5. Реком. отгрузка = `max(0, спрос/день × целевые дни − доступно)`

### 5) Сезонность
Коэффициент сезона берется из месячного плана по артикулам:

`коэф. сезона = план/день текущего месяца ÷ средний план/день по доступным месяцам`

Есть ручной override в форме.

### 6) Формы
- **Быстрое внесение данных** — в пути / производство / закупка / target days / seasonality override
- **Заказ товара** — заявка на отгрузку / производство / закупку
- **Экспорт / импорт JSON**
- экспорт CSV по:
  - запасам
  - ручным данным
  - заявкам

## Публикация
1. Заменить содержимое репозитория файлами из этой папки.
2. Дождаться обновления сайта.
3. Открыть:
   - основной сайт: `index.html`
   - модуль Нины: `nina.html`
4. Сделать `Ctrl+F5`.

## Важно
Ручные данные на `nina.html` пока сохраняются **локально в браузере**.  
Чтобы переносить их между устройствами — используйте экспорт / импорт JSON.
