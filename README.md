# Tekstilno portal v23 shared ready

Что внутри:
- `index.html` — понятный каркас портала: Сегодня / Задачи менеджеров / Кластера и отгрузка / Продажи и маржа / Журнал сдачи.
- `nina.html` — полноэкранная матрица Нины.
- `shared-storage.js` — общее хранилище: `session-preview` без localStorage или `supabase` для общего журнала.
- `backend/portal_shared_storage_supabase.sql` — минимальная схема для общего сохранения отчётов, задач и заявок.

Что изменено в этой сборке:
- исправлен переключатель оборачиваемости 7 / 14 / 28;
- WB артикул добавлен в портал и в матрицу Нины;
- портал стал понятнее: контуры вынесены в отдельные плитки;
- localStorage больше не является основным способом хранения;
- журнал отчётности и заявки готовы к общему сохранению через Supabase.
