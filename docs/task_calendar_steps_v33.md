# V33 — календарь выполнения задач по датам

Что добавлено:
- календарь рабочих дат в разделе `Задачи менеджеров`;
- быстрый переход по прошлым / текущим / будущим датам;
- в `Журнале сдачи` появилась сводка по датам;
- используется уже существующий Supabase backend, новый SQL не нужен.

Как работать:
1. Откройте `Задачи менеджеров`.
2. Выберите площадку `WB` или `Ozon`.
3. Нажимайте по карточкам дат в календаре.
4. Для руководителя доступны кнопки:
   - `Зафиксировать задачи на дату`
   - `Скопировать план с пред. даты`
5. Менеджер открывает нужную дату и заполняет статус + комментарий.
6. В `Журнале сдачи` видно выполнение по датам.

Легенда:
- `Готово` — все SKU на дату закрыты;
- `В работе` — по дате есть движение;
- `План` — задачи на дату уже поставлены, но ещё не двигаются;
- `Хвосты` — прошлые даты, где задачи не закрыты;
- `Нужна помощь` — по дате есть хотя бы один статус `need_help`.
