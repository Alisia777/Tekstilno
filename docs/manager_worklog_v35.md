# Manager worklog v35

Что добавлено:
- в разделе `Задачи менеджеров` появился блок `Доп. строка работы по артикулу`;
- менеджер может выбрать любой артикул из полной матрицы WB/Ozon;
- каждая запись сохраняется отдельно в `portal_task_comments` с `comment_type = 'manager_worklog'`;
- предыдущие записи не затираются;
- запись сразу попадает в общую сводную внизу в разделе `Журнал сдачи`.

Как пользоваться:
1. Выбрать рабочую дату сверху или прямо в блоке `Дата работы`.
2. Вставить sellerArticle или WB артикул.
3. Выбрать тип работы и статус.
4. Написать, что сделано.
5. Нажать `Добавить строку в общую сводную`.

Важно:
- если запись добавляется за прошлую или будущую дату, портал автоматически переключится на эту дату;
- фиксированные 20 SKU остаются как основной список;
- доп. строки работы идут отдельным append-only журналом.
